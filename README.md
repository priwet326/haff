Реализация алгоритма сжатия методом Хаффмена
Как использовать
Скомпилируйте main.cpp с помощью Makefile(установите make если не установлен)
Скомпилированный файл будет иметь имя haffmen
------
Есть флаги которые можно использовать
  Для простого кодирования:
    ./haffmen file_name.file_format
    на выходe encoded_file_name.bin
  Для декодирования
    ./haffmen -d encode_file_name.bin
    на выходе decode_file_name
  Остаётся только добавить формат файла заранее проверив его с помощью команды 
    file decode_file_name
  из вывода надо будет взять формат файла file_format
  и добавить формат с помощью
    mv decode_file_name.format_file
------
Также имеется вспомогательная функция рассчёта энтропии файла
./haffmen -e file
